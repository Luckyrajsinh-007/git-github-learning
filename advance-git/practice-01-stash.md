# Git Stash – Practice 01

## What is Git Stash?
Git stash temporarily saves uncommitted changes so you can switch branches
without committing unfinished work.

---

## When to Use Git Stash
- You are working on a feature
- An urgent bug fix comes
- You don’t want to commit incomplete code

---

## Commands Used

```bash
git stash
git stash list
git stash apply
git stash pop
git stash drop

---
